# Glossary

> Update this file whenever new canonical terms are introduced.

| Term | Meaning |
|---|---|
| AI factory | This repository viewed as a GitHub-native system for generating and maintaining game audio assets |
| Asset plan | The project-level statement of which audio assets should exist |
| Generation request | One concrete instruction to generate or regenerate a specific asset |
| Style family | A safe, abstract inspiration bucket rather than a copyrighted work copy |
| Provenance | Metadata about prompt, seed, backend, outputs, and review history |
| Review status | Approval state such as `draft`, `approved`, `revise`, `rejected` |
| Canonical filename | Stable path/name expected by a consuming pipeline or mapping layer |
| Override asset | A manually replaced asset that supersedes a generated one |
| Loop-safe | Suitable for seamless looping without obvious seam/click |
| Downstream repo | A consumer such as `GameRewritten` |
