# ADR-002: Use style families instead of direct copyrighted imitation

- Status: Accepted
- Date: 2026-05-14

## Context

The desired inspiration targets include major commercial franchises. Literal copying would be risky and would also make future prompting ambiguous.

## Decision

Use abstract style families, mood language, instrumentation guidance, and production descriptors. Do not request or aim for exact melodies, recordings, motifs, or signature sound matches.

## Consequences

- Prompting should use descriptors like `melancholic sci-fi field theme` instead of copyrighted track references.
- Review should judge emotional fit and gameplay utility, not similarity to a specific song.
- Documentation should keep this boundary explicit for future agents.
