# Style Families

> Update this file whenever style guidance changes.

## Safety rule

The repo may target emotions and production qualities associated with famous franchises, but it must not request or reproduce copyrighted compositions or signature recordings.

## Approved style families

| Family ID | Description | Safe inspiration zone |
|---|---|---|
| `heroic-sci-fantasy` | noble adventure, orchestral motion, layered harmonic uplift | large-scale JRPG field/battle energy |
| `melancholic-techno-fantasy` | bittersweet harmony, synthetic shimmer, reflective pacing | introspective sci-fantasy melancholy |
| `sacred-mythic` | choir, bells, solemn suspended harmony | temples, prayer, fate, revelation |
| `industrial-boss-pressure` | aggressive rhythm, tension, metallic pulse | boss escalation and threat |
| `warm-town-nostalgia` | gentle acoustic/piano/light ensemble | safe hub areas and downtime |
| `mythic-space-opera` | broad brass, cinematic propulsion, high drama | space-opera scale and heroism |
| `mystery-dungeon-ether` | eerie pads, sparse percussion, uncertain harmony | dungeons, ruins, anomaly spaces |

## Use these descriptors in prompts

Prefer:

- emotional tone
- gameplay function
- instrumentation
- pacing
- density
- loop behavior

Example:

- `loopable field exploration theme, melancholic-techno-fantasy, piano + strings + synth shimmer, medium tempo`

Avoid:

- `make this sound like [exact copyrighted track]`
- `copy the boss theme from [game]`
- `use the same melody as [franchise cue]`

## Priority guidance

Music and SFX style work are high priority. Voice style work is lower priority and should not block soundtrack/system SFX improvements.
