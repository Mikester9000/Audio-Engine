# Emotion and Mood Matrix

> Update this file when new mood categories become canonical.

## Music mapping

| Gameplay state | Primary moods | Secondary moods | Candidate style families |
|---|---|---|---|
| Main menu | wonder, anticipation | melancholy, calm | `melancholic-techno-fantasy`, `heroic-sci-fantasy` |
| Field exploration | adventure, openness | nostalgia, curiosity | `heroic-sci-fantasy`, `warm-town-nostalgia` |
| Town | comfort, life | warmth, bustle | `warm-town-nostalgia` |
| Dungeon | tension, mystery | dread, discovery | `mystery-dungeon-ether`, `sacred-mythic` |
| Standard battle | urgency, momentum | threat, confidence | `heroic-sci-fantasy`, `industrial-boss-pressure` |
| Boss battle | danger, grandeur | desperation, myth | `industrial-boss-pressure`, `mythic-space-opera` |
| Victory | relief, triumph | uplift, pride | `heroic-sci-fantasy` |
| Shop/menu/save | clarity, rest | optimism, ritual | `warm-town-nostalgia`, `sacred-mythic` |

## SFX mapping

| Category | Emotional target | Notes |
|---|---|---|
| UI confirm | clarity | fast, readable, non-annoying |
| UI cancel | soft rejection | lower energy than confirm |
| Sword hit | impact | readable transient, not muddy |
| Magic cast | anticipation | should distinguish schools/elements |
| Fire spell | aggressive heat | short bright burst plus tail |
| Ice spell | brittle cold | crystalline transient |
| Thunder | power | sharp crack + low body |
| Quest complete | reward | short celebratory lift |
| Ambient wind/rain | immersion | non-fatiguing loop |
